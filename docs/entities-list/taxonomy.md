# Nepal Government Entity Taxonomy

This document defines the entity subtypes for classifying all government entities in Nepal's federal democratic republic structure. All entities use the type `government` with specific subtypes for categorization. This taxonomy is used by the Jawafdehi platform for consistent categorization and accountability tracking.

## Usage Guidelines

### Entity Classification Rules

1. **Single Entity Type**: All government entities use the type `government`

2. **Hierarchical Structure**: Entities are classified based on their administrative level:
   - Federal > Provincial > District > Local > Ward

3. **Functional Classification**: Within each level, classify by function:
   - Executive, Legislative, Judicial, Constitutional, Administrative

4. **Consistent Naming**: Use standardized subtypes for database consistency

5. **Bilingual Support**: Always provide both English and Nepali names where applicable

### Example Entity Records

```json
{
  "name": "Ministry of Home Affairs",
  "name_nepali": "गृह मन्त्रालय",
  "entity_type": "government",
  "subtype": "ministry_home_affairs",
  "level": "federal",
  "parent_entity": "Government of Nepal",
  "jurisdiction": "Nepal",
  "location": "Singha Durbar, Kathmandu"
}
```
