## Welcome to Jawafdehi

Welcome to the Jawafdehi team! Whether you're joining as a Case Documentation Intern or a Software Engineering Intern, we're excited to have you contribute to Nepal's transparency and accountability platform. Jawafdehi (https://jawafdehi.org) is an open database of corruption cases in Nepal. Our mission is to help citizens access information about allegations of corruption and misconduct by public entities. We believe that transparency is essential for accountability in governance. This document will help you get started and understand how we work.

## Getting Started

1. **Join Discord** - Use a name you're comfortable being called
2. **Message the team lead** - Get the `intern` tag for access to relevant channels
3. **Review this document** - Familiarize yourself with essential skills and expectations
4. **Set up your development environment** - Instructions will be provided

## Essential Skills

We will provide training for all of these skills, so don't worry if you're not familiar with them yet.

### Git
- **Branching**: Create feature branches, understand branch workflows
- **Commits**: Write clear, meaningful commit messages
- **Pull requests**: Submit and review code changes

### Code Editor
- **VS Code** or **Kiro**: Our recommended development environments
- Familiarize yourself with extensions and shortcuts
- Learn to navigate large codebases efficiently

### Command Line
- **Basic navigation**: cd, ls, pwd
- **File operations**: cp, mv, rm, mkdir
- **Running commands**: Execute scripts and development servers
- **Package managers**: Poetry (Python), Bun (TypeScript)
